# 教員向け運用マニュアル

## 1. はじめに
教員用アカウントでは、担当クラスの出欠確認、申請の承認、時間割の管理を行うことができます。

## 2. 出欠確認

### 2.1 今日の出欠
1. ダッシュボードまたは「出欠管理」メニューから、担当クラスを選択します。
2. 当日の出欠状況（出席、欠席、遅刻）が一覧表示されます。
3. 必要に応じて、手動でステータスを変更・修正できます。

### 2.2 履歴の確認
- 過去の日付を選択して、出欠履歴を確認できます。
- 学生ごとの月間出欠率なども確認可能です。

## 3. 申請の承認

### 3.1 承認ワークフロー
学生から提出された「欠席届」「遅刻届」などを承認します。
1. 「承認管理」メニューを選択します。
2. 「未承認」タブに、対応が必要な申請が表示されます。
3. 申請内容を確認し、「承認」または「却下」ボタンをクリックします。
4. 承認すると、自動的に出欠データに反映されます。

## 4. 時間割管理

### 4.1 時間割の登録
1. 「時間割」メニューを選択します。
2. 担当クラスを選択し、曜日・時限ごとに授業を登録します。
3. Excelファイルによる一括インポートも可能です。

### 4.2 授業QRコードの発行
授業ごとに一時的なQRコードを発行して出席を取る場合に使用します。
1. 時間割から対象の授業を選択します。
2. 「出席用QR発行」をクリックします。
3. スクリーンにQRコードを表示し、学生にスキャンさせます。
   - このQRコードは数分間のみ有効です。

## 5. クラス管理

### 5.1 学生の管理
- 担当クラスの学生一覧を確認できます。
- 学生の基本情報や保護者連絡先などを参照できます。
