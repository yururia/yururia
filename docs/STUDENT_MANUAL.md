# 学生向け運用マニュアル

## 1. はじめに
このシステムを使って、日々の出席登録や欠席の連絡を行うことができます。

## 2. 出席登録

### 2.1 QRコードスキャン
教室に入室したら、スマートフォンで出席登録を行います。
1. アプリにログインします。
2. 「QRスキャン」ボタンをタップします。
3. 教室に掲示されている、またはスクリーンに表示されたQRコードを読み取ります。
4. 「出席が記録されました」と表示されれば完了です。
   - **注意**: 校内ネットワーク（Wi-Fi）に接続している必要があります。

## 3. 申請（欠席・遅刻）

### 3.1 欠席・遅刻の連絡
病気や用事で欠席・遅刻する場合は、アプリから申請します。
1. 「申請」メニューを選択します。
2. 「新規申請」ボタンをタップします。
3. 種類（欠席/遅刻/早退/公欠）を選択します。
4. 日付と理由を入力し、送信します。
5. 教員が承認すると、通知が届きます。

## 4. 時間割とカレンダー

### 4.1 時間割の確認
- 「時間割」メニューから、今週の授業スケジュールを確認できます。

### 4.2 カレンダー
- 「カレンダー」メニューから、自分の出欠履歴や今後の予定を確認できます。
- 承認された欠席などもここに表示されます。

## 5. プロフィール設定
- パスワードの変更や、連絡先情報の確認ができます。
