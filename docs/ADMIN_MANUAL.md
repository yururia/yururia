# 管理者向け運用マニュアル

## 1. はじめに
本システムは、学校および企業の出欠管理を統合的に行うプラットフォームです。
管理者は、組織全体の設定、ユーザー管理、およびセキュリティ設定を行う権限を持ちます。

## 2. ユーザー管理

### 2.1 ユーザーの登録
1. 管理画面の「ユーザー管理」メニューを選択します。
2. 「新規登録」ボタンをクリックします。
3. 氏名、メールアドレス、役割（管理者/教員/従業員）を入力し、保存します。
   - **管理者**: 全ての機能にアクセス可能
   - **教員**: 担当クラスの管理、承認が可能
   - **従業員**: 一般的な出欠記録のみ可能

### 2.2 学生の登録
1. 「学生管理」メニューを選択します。
2. 個別に登録するか、CSVインポート機能を使用して一括登録します。
3. 学生IDとカードID（ICカード連携用）を紐付けます。

## 3. クラス・グループ管理

### 3.1 クラスの作成
1. 「クラス管理」メニューを選択します。
2. 「新規作成」をクリックし、クラス名、学年、年度を入力します。
3. 作成後、詳細画面から「担当教員」と「所属学生」を割り当てます。

### 3.2 担当教員の割り当て
- クラス詳細画面の「担当教員」タブで、教員を追加します。
- 担当教員のみが、そのクラスの出欠承認や編集を行えます。

## 4. QRコード管理

### 4.1 場所ベースQRコードの発行
教室や会議室に設置する固定QRコードを発行します。
1. 「QRコード管理」メニューを選択します。
2. 「場所用QR発行」をクリックします。
3. 場所名（例: 「301教室」）を入力し、有効期限を設定して発行します。
4. 生成されたQRコードを印刷して掲示します。

## 5. セキュリティ設定

### 5.1 IPアドレス制限
不正なアクセスを防ぐため、校内/社内ネットワークからのアクセスのみを許可します。
1. 「セキュリティ設定」メニューを選択します。
2. 「許可IP範囲」に、組織のグローバルIPアドレス範囲（CIDR形式）を追加します。
3. QRスキャン時にこのIP制限が適用されます。

## 6. 監査ログ
システムの操作履歴を確認できます。
- 「監査ログ」メニューから、誰がいつ何をしたかを確認できます。
- 不審な操作があった場合の追跡に使用します。
